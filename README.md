# Ticket Monster

Website for CS4973 (Abolition Tech) Community Project deliverable.

**Goals**
1. Familiarize my community on Ticketmaster’s background
2. Provide step-by-step on successfully purchasing tickets within the existing system (and practice with simulation)
3. Detail workarounds/alternative ticket-purchasing options
4. Inform on long-term activism

**Future updates**
1. Incorporate more simulation cases (credit card details not saved, sign-up/login).
2.  Improve accessibility by changing the font colors to contrast more.
3.  Implement a dark/light mode toggle.
4.  Add panels to the alternatives/actvism/step-by-step pages for community members to post and share their own tips.
